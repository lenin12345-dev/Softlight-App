# Softlight Auth Screens

This project implements the sign-in, sign-up, and forgot-password screens based on the provided Softlight mobile design using **React** and **Material UI (MUI)**.

## Tech stack

- React (Vite + TypeScript)
- Material UI (`@mui/material`)
- Vite for bundling

## Features

- Responsive authentication screens (Sign-In, Sign-Up, Forgot Password)

- Form validation for email and password fields

- Custom GradientButton for primary actions

##  Limitations

- No backend integration (forms only validate on frontend)

- Basic form validation; some edge cases may not be handled

- Direct file opening (file://) does not work due to CORS restrictions

- No social login options implemented yet
